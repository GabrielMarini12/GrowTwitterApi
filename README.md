# GrowTwitterApi

API do prjeto Twitter (Growdev)

# Tecnologia

- TypeScript

# Banco de Dados

- Prisma

# Começando
- Dependências:
  - iniciar projeto
    - npm init- y
  - instalando dependências
    - npm install -D typescript ts-node-dev
    - npm install -D @types/node
    - npm i express
    - npm i -D prisma
    - npm i @prisma/client
    - npx prisma init --datasource-provider postgresql
    - npm i express dotenv
    - npm i -D @types/express
    - npx tsc --init
    - npx tsc
